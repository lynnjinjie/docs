Markdown教程

# 简介

Markdown是一种轻量级的标记语言,使用易读易写的纯文本格式编写文档，可转换成的 XHTML（或者 HTML）文档。

# 应用

主流的网站和软件都支持Markdown的格式书写文字, 如: github,印象笔记, 简书等.

# 语法使用

# 一. 标题

示例:

# 这是一级标题

## 这是二级标题

### 这是三级标题

#### 这是四级标题

##### 这是五级标题

###### 这是六级标题

效果如下:

# 这是一级标题

# 这是二级标题

# 这是三级标题

# 这是四级标题

# 这是五级标题
# 这是六级标题

# 二. 字体

示例:

_斜体_
**粗体**
**_粗斜体_**
~~删除线~~
<u>下划线</u>
`文字高亮`

效果如下:

斜体 粗体 粗斜体 删除线 下划线 文字高亮

# 三. 引用

示例:

> 引用
>
> > 引用
> >
> > > 引用

效果如下:

引用

引用

引用

# 四. 分割线

示例:

---

效果如下:

# 五. 图片

示例:

![图片alt](图片地址 '图片title')

图片 alt: 显示在图片下方的文字,通常是对图片的说明
图片 title: 鼠标放在图片上显示的内容, 可加可不加
示例:
![首页banner](https://cdn.jsdelivr.net/gh/lynnjinjie/images_store/img/banner.jpg 'banner图')

效果如下:

如果要使用一个链接能够随时使用图片,可以使用图床

# 六. 链接

# 超链接

示例:

[百度](https://www.baidu.com)

效果如下: 百度 (opens new window)

注意: 如果要在新标签页打开需要使用 html 的 a 标签

<a href="https://www.baidu.com" target="_blank">百度</a>

效果如下: 百度

# 变量链接

示例:

先定义好链接变量 [baidu]: https://www.baidu.com

使用: [百度][baidu]

效果如下: 百度 (opens new window)

# 锚点链接

markdown 文件中每个标题都是一个锚点

# 七. 列表

# 无序列表

示例:

- 列表内容

* 列表内容

- 列表内容

效果如下:

  • 列表内容
  • 列表内容
  • 列表内容

# 有序列表

示例:

1. 列表内容
2. 列表内容
3. 列表内容

效果如下:

  1. 列表内容
  2. 列表内容
  3. 列表内容

# 折叠列表

示例:

<details>
  <summary>点击打开折叠的内容</summary>
  折叠内容
  <ul>
    <table>
      <thead>
        <td>1</td>
        <td>2</td>
        <td>3</td>
      </thead>
      <tr>
        <td>qqq</td>
        <td>qqq</td>
        <td>qqq</td>
      </tr>
      <tr>
        <td>www</td>
        <td>www</td>
        <td>www</td>
      </tr>
    </table>
  </ul>
</details>

效果如下:

details 里只能使用 html 标签,如 ul,li, table

点击打开折叠的内容 折叠内容
    1 2 3
    qqq qqq qqq
    www www www

# 复选框列表

示例:

- [x] 1
- [x] 2
- [x] 3

效果如下:

注: 部分线上无法显示

  • [x] 1
  • [x] 2
  • [x] 3

# 嵌套列表

注: 子节点需要缩减 3 个

示例:

1. 第一项

   - 第一项的第一个子元素
   - 第一项的第二个子元素
     - 第三层嵌套

效果如下:

  1. 第一项
    • 第一项的第一个子元素
    • 第一项的第二个子元素
      • 第三层嵌套

# 八. 代码

使用三个反引号```包裹代码块,也可指定语言,会有高亮效果

示例:

​```javascript

function test() {
alert('test')
}

​```

效果如下:

function test() {
  alert('test')
}

# 九. 表格

使用 |来分隔不同的单元格, -来区分表头和表体

示例:

| 表头   | 表头   | 表头   |
| ------ | ------ | ------ |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

效果如下:

表头 表头 表头
单元格 单元格 单元格
单元格 单元格 单元格

# 对齐方式

  • -: : 右对其
  • :- : 左对其
  • :-: : 居中对其 示例:
| 左对其 |  居中  | 右对其 |
| :----- | :----: | -----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

效果如下:

左对其 居中 右对其
单元格 单元格 单元格
单元格 单元格 单元格
